5632. There are \(2^{11}\) strings that start with 011, and another \(2^{12}\) which end with 01 (we choose 1 or 0 for 12 bits). However, we count the strings that start with 011 and end with 01 twice - there are \(2^{9}\) such strings. So using PIE, we have \(2^{11} + 2^{12} - 2^{9} = {5632}\)
