Item 1.2.3.c.
For each of the 4 elements from \(A\text{,}\) we must decide yes or no on whether to include them in the subset. However, for the odd numbers, we only have one choice: no. So there are only 2 elements we have two choices for, so the answer is \(2^{2} = {4}\text{.}\) (Note, if you wish to exclude the empty set - it does not contain odd numbers, but no evens either - then you could subtract 1).
