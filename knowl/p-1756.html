Now all the ways to distribute the 4 units to the four \(y_i\) variables can be found using stars and bars, specifically 4 stars and 3 bars, so \({7 \choose 4}\) ways. But this includes the ways that one or more \(y_i\) variables can be assigned more than 3 units. So subtract using PIE to get \(\displaystyle{ {7 \choose 3} -\left( {4\choose 1} {3 \choose 3}-{4\choose 2} {-1 \choose 3} + \dots \right) }\)
