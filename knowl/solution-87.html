Solution 1.2.5.1.

  1. We can think of each row as a 8-bit string of weight 4 (since of the 8 coins, we require 4 to be pennies). Thus there are \({8 \choose 4} = {70}\) rows possible. Each row requires 8 coins, so if we want to make all the rows at the same time, we will need 560 coins, half of them nickels and half of them pennies.
  2. Now there are \(2^{8} = 256\) rows possible, which is also \({8 \choose 0} + {8\choose 1} + {8 \choose 2} +\dots + {8 \choose 8}\text{,}\) if you break them up into rows containing 0, 1, 2, etc. pennies. Thus, since there are 8 coins in each row, we need \(8 \cdot 256 = {2048}\) total coins.
