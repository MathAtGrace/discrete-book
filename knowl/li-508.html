Item 1.1.4.a.
There are 1048576 6-digit hexadecimals in which the first digit is an E (one for each choice of the remaining digits). Similarly, there are 1048576 hexadecimals in which the first digit is an F. We want the union of these two disjoint sets, so there are \(1048576 + 1048576 = 2\cdot1048576=2097152\) 6-digits hexadecimals in which the first digit is either an E or an F.
