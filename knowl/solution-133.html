Solution 1.5.10.1.

  1. \({8\choose 4}= {70}\text{.}\) Note that a strictly increasing function is automatically injective. So the 4 outputs must all be different. So let’s first pick which 4 outputs we will use: there are \({8 \choose 4}\) ways to do this. Now how many ways are there to assign those outputs to the inputs \(1\) through 4? Only one way, since there is only one way to arrange numbers in increasing order.
  2. \({11 \choose 4}\text{.}\) This is in fact a stars and bars problem. The stars are the 4 inputs and the bars are the 7 spots between the 8 possible outputs. Think of it this way - we will specify \(f(1)\text{,}\) then \(f(2)\text{,}\) then \(f(3)\text{,}\) and so on in that order. Start with the possible output 0. We can use it as the output of \(f(1)\text{,}\) or we can switch to 1 as a potential output. Say we put \(f(1) = 1\text{.}\) Now we are at 1 (can’t go back to 0). Should \(f(2) = 1\text{?}\) If yes, then we are putting down another star. If no, put down a bar and switch to 2. Maybe you switch to 3, then assign \(f(2) = 3\) and \(f(3) = 3\) (two more stars) before switching to 4 as a possible output. And so on.
