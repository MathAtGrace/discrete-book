Now there are \(2^{8} = 256\) rows possible, which is also \({8 \choose 0} + {8\choose 1} + {8 \choose 2} +\dots + {8 \choose 8}\text{,}\) if you break them up into rows containing 0, 1, 2, etc. pennies. Thus, since there are 8 coins in each row, we need \(8 \cdot 256 = {2048}\) total coins.
