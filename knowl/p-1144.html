624. There are \({11 \choose 3} = 165\) strings of weight 5 which start with 011, and another \({12 \choose 4} ={495}\) which end with 01. We have over counted again - there are weight 5 strings which both start with 011 and end with 01, in fact \({9 \choose 2} ={36}\) of them. So all together we have \(165+{495}-{36} = {624}\) strings.
