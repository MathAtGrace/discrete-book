We can think of each row as a 8-bit string of weight 4 (since of the 8 coins, we require 4 to be pennies). Thus there are \({8 \choose 4} = {70}\) rows possible. Each row requires 8 coins, so if we want to make all the rows at the same time, we will need 560 coins, half of them nickels and half of them pennies.
