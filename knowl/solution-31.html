Solution 0.3.9.1.

  1. \(A \cap B\) will be the set of natural numbers that are both at least 3 and less than 16, and even. That is, \(\renewcommand{\d}{\displaystyle}\newcommand{\N}{\mathbb N}\newcommand{\st}{:}\renewcommand{\v}{\vtx{above}{}}A \cap B = \{x \in \N \st 3 \le x \lt 16 \wedge x \text{ is even}\} = {\left\{4,6,8,10,12,14\right\}}\text{.}\)
  2. \(A \setminus B\) is the set of all elements that are in \(A\) but not \(B\text{.}\) So this is \(\renewcommand{\d}{\displaystyle}\newcommand{\N}{\mathbb N}\newcommand{\st}{:}\renewcommand{\v}{\vtx{above}{}}\{x \in \N \st 3 \le x \lt 16 \wedge x \text{ is odd}\} = {\left\{3,5,7,9,11,13,15\right\}}\text{.}\)
    Note this is the same set as \(A \cap \overline{B}\text{.}\)
