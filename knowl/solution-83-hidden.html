1. \(2^{15} = 32768\) subsets. We need to select yes/no for each of the 15 elements.
  2. \(2^{12} = 4096\) subsets. We need to select yes/no for each of the remaining 12 elements.
  3. \(2^{15} - 2^{7} = 32640\) subsets. We subtract the number of subsets which do not contain any odd numbers (\(2^{7}\)-select yes or no for each even element) from the total number of possible subsets.
  4. \({7 \choose 1}\cdot 2^{8} ={1792}\) subsets. First pick the even number. Then say yes or no to each of the odd numbers.
