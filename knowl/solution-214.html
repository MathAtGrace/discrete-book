Solution 2.3.2.1.

\(a_n = {3n^{2}+6n-4}\text{.}\) Here we know that we are looking for a quadratic because the second differences are constant. So \(a_n = an^2 + bn + c\text{.}\) Since \(a_0 = {-4}\) know \(c= {-4}\text{.}\) So just solve the system
\(\begin{aligned} {5} \amp = a + b + {-4}\\ {20} \amp = 4a + 2b + {-4} \end{aligned}\)
to find \(a = {3}\) and \(b = {6}\text{.}\)
