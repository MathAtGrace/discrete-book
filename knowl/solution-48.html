Solution 0.4.1.1.

  1. \(f(2) = 5\text{,}\) since 5 is the number below 2 in the two-line notation.
  2. Such an \(n\) is \(n= 4\text{,}\) since \(f(4) = 2\text{.}\) Note that 4 is above a 2 in the notation.
  3. \(n = 3\) has this property. We say that 3 is a fixed point of \(f\text{.}\) Not all functions have such a point.
  4. Such an element is 4 (in fact, that is the only element in the codomain that is not in the range). In other words, 4 is not the image of any element under \(f\text{;}\) nothing is sent to 4.
