\({8 \choose 6} = {28}\) functions. Since the function must be injective and increasing, we just need to select the 6 distinct elements of the range from the 8 elements of the codomain. Once selected, we must put the smallest as the image of 1, the next smallest as the image of 2, and so on (doing this does not increase the number of functions, since there is one choice for how this event can occur).
