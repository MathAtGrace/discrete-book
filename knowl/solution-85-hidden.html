1. There are \({16}\) subsets. This is \(2^4\text{,}\) which makes sense because we are deciding yes or no on whether to include each element of \(A\) in the subset.
  2. Of the 4 elements in \(A\text{,}\) we must choose 3 of them to be in the subset. So \({4 \choose 3} = {4}\text{.}\)
  3. For each of the 4 elements from \(A\text{,}\) we must decide yes or no on whether to include them in the subset. However, for the odd numbers, we only have one choice: no. So there are only 2 elements we have two choices for, so the answer is \(2^{2} = {4}\text{.}\) (Note, if you wish to exclude the empty set - it does not contain odd numbers, but no evens either - then you could subtract 1).
  4. Count the number of subsets with each possible even cardinality:
    \(\displaystyle{ {4 \choose 0} + {4 \choose 2} + \dots... = {8} }\)
