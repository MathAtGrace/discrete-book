\(a_n = {3n^{2}-4n}\text{.}\) Here we know that we are looking for a quadratic because the second differences are constant. So \(a_n = an^2 + bn + c\text{.}\) Since \(a_0 = 0\text{,}\) we know \(c= 0\text{.}\) So just solve the system
\(\begin{aligned} {-1} \amp = a + b\\ {4} \amp = 4a + 2b \end{aligned}\)
to find \(a = 3\) and \(b = -4\text{.}\)
