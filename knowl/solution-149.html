Solution 1.6.4.1.

The easiest way to solve this is to first distribute the minimum amount of units to each variable (3) and then count the solutions to \(y_1 + y_2 + y_3 + y_4 = 4\) with \(0 \le y_i \le 3\text{.}\) By taking \(x_i = y_i+3\text{,}\) each solution to this new equation corresponds to exactly one solution to the original equation.
Now all the ways to distribute the 4 units to the four \(y_i\) variables can be found using stars and bars, specifically 4 stars and 3 bars, so \({7 \choose 4}\) ways. But this includes the ways that one or more \(y_i\) variables can be assigned more than 3 units. So subtract using PIE to get \(\displaystyle{ {7 \choose 3} -\left( {4\choose 1} {3 \choose 3}-{4\choose 2} {-1 \choose 3} + \dots \right) }\)
The \({4 \choose 1}\) counts the number of ways to pick one variable to be over-assigned, the \({3 \choose 3}\) is the number of ways to assign the remaining units to the 4 variables, etc.
