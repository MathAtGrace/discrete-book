Solution 2.3.4.1.

\(a_n = {n^{3}-4n^{2}+8n+1}\text{.}\) Here we know that we are looking for a cubic because the third differences are constant. So \(a_n = an^3 + bn^2 + cn + d\text{.}\) We can work backwards to find that \(a_0 = {1}\text{,}\) so we know \(d= {1}\text{.}\) Then solve the system,
\(\begin{aligned} {6} \amp = a + b + c + {1}\\ {9} \amp = 8a + 4b + 2c + {1}\\ {16} \amp = 27a + 9b + 3c + {1}\\ \end{aligned}\)
to find \(a = {1}\text{,}\) \(b = {-4}\text{,}\) and \(c = {8}\text{.}\)
