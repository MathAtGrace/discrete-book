  2. Without any restriction, there would be \({37\choose 22}\) ways to distribute the stars. Now we must use PIE to eliminate all distributions in which one or more student gets more than one star:
\(\displaystyle{ {37 \choose 22} - \left[{23 \choose 1}{35 \choose 22} - {23\choose 2}{33 \choose 22} + {23\choose 3}{31 \choose 22} - \cdots \right] = 490314. }\)
